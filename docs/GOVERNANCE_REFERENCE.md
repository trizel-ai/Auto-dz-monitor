# Governance Reference

This project adheres to the accepted standards for open source project governance. For a detailed explanation and canonical reference, see:

- [The Open Source Guides: Leadership and Governance](https://opensource.guide/leadership-and-governance/)
- [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
- [Maintainer Roles and Responsibilities](https://github.com/opensource/maintainer-responsibilities)

These documents and resources outline the decision-making processes, escalation paths, roles and responsibilities, and general best practices followed by this project.
